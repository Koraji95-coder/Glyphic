## Color Themes

Each mockup includes a **theme switcher** (top-right corner) with 5 options:

| Theme | Accent Color | Vibe |
|-------|-------------|------|
| **Cyber Indigo** (default) | `#7c6df0` purple-blue | Modern, focused, developer |
| **Emerald** | `#3dd68c` green | Fresh, calm, nature |
| **Amber** | `#e0a050` warm gold | Classic, warm (close to current) |
| **Rose** | `#e06088` pink-red | Bold, creative, energetic |
| **Light** | `#5850d8` deep indigo | Clean light mode |

## Design Direction

**Obsidian-like but better** — information-dense and power-user focused, with modern polish:

- Deeper, more layered dark backgrounds (`#0e0f14` base vs current `#111113`)
- Gradient accent brand icon instead of flat color
- Pill-grouped toolbar buttons with subtle card backgrounds
- Active items with left accent border indicator
- Richer typography hierarchy (DM Serif Display for titles, DM Sans for body)

12. **Cursor Position** — Line:Column in status bar (mockup 02, 03, 05)

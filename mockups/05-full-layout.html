Binary Search Trees

A BST is a binary tree where each node's left subtree contains only nodes with keys less than the node's key, and the right subtree contains only keys greater than the node's key.

Key Operations

  • search(key) — O(log n) average, O(n) worst
  • insert(key) — O(log n) average
  • delete(key) — O(log n) average
Exam tip: Always mention worst-case for unbalanced trees degenerating into linked lists.

⏱ 05:22 · 2:35 PM

AVL Trees

Self-balancing BST where the balance factor of any node is at most 1. Uses rotations to maintain balance.

⏱ 12:34 · 2:42 PM

Hash Tables

Hash tables provide O(1) average-case lookup. Collision strategies include chaining and open addressing.
